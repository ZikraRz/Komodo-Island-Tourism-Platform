# Komodo Island Tourism Platform

The **Komodo Island Tourism Platform** is a web-based tourism system designed to help travelers explore Komodo Island, Indonesia. This platform provides information about travel destinations, tour packages, accommodation options, and trip bookings to support visitors in planning their travel experience efficiently.

## Features
- Destination Information – Detailed descriptions of tourist attractions
- Tour Packages – Various packages with pricing and facilities
- Booking System – Allows users to book tours
- User-Friendly Interface – Simple and easy to navigate
- Responsive Design – Accessible on desktop and mobile devices

## Tech Stack
- Frontend: HTML, CSS, JavaScript    
- Tools: VS Code, Git, GitHub  

## Installation
1. Clone the repository  
   git clone https://github.com/ZikraRz/Komodo-Island-Tourism-Platform.git

2. Navigate to the project directory  
   cd Komodo-Island-Tourism-Platform

3. Open the project using a local server or browser
